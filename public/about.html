关于本站

项目简介

"一战缘起"是一个交互式历史探索网站，旨在通过现代化的网页技术， 深入浅出地展现第一次世界大战爆发的复杂历史背景和深层原因。

本站采用响应式设计，支持桌面端和移动端访问， 通过交互式图表、时间线和可视化内容，让历史学习变得更加生动有趣。

技术栈

  • 前端框架: HTML5, CSS3, JavaScript
  • 样式框架: Tailwind CSS
  • 图表库: Chart.js
  • 字体: Noto Sans SC (Google Fonts)
  • 部署平台: Cloudflare Workers
  • 域名: nssa.io
